# RoseGuard Monitor

A full-stack health monitoring application that collects real-time biometric data from an ESP32 microcontroller via Bluetooth (SpO2, body temperature, heart rate), processes it into health KPIs, and displays live readings on a web and mobile app.

## Tech Stack

| Layer | Technology |
|---|---|
| Hardware | ESP32 (BLE) |
| Backend | ASP.NET Core Web API (.NET 8) |
| Real-time | SignalR |
| Database | PostgreSQL + EF Core |
| PDF Reports | QuestPDF |
| Frontend | Vue 3 + Vite + Pinia + ApexCharts |
| Mobile | Capacitor (Android / iOS) |

## Project Structure

```
/backend     ← ASP.NET Core Web API
/frontend    ← Vue 3 + Vite
/android     ← generated by Capacitor
/ios         ← generated by Capacitor
```
